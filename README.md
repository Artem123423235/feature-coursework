## 📊 Финансовый аналитик - Курсовая работа

### 🎯 Описание проекта
Приложение для анализа финансовых данных с веб-интерфейсом.

### 🚀 Что реализовано
- **Модуль services**: Работа с API курсов валют, расчеты кешбэка и инвестиций
- **Модуль reports**: Генерация отчетов по категориям расходов
- **Модуль views**: Веб-интерфейс на Flask
- **Тесты**: Полное покрытие тестами всех модулей

### 📋 Функционал
1. 📈 Получение курсов валют с внешнего API
2. 💰 Расчет выгодных категорий для кешбэка
3. 🏦 Формирование инвестиционных рекомендаций
4. 📊 Генерация финансовых отчетов
5. 🌐 Веб-интерфейс для взаимодействия

### 🧪 Тестирование
